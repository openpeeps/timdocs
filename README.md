## About
This repo is used for the documentation of Tim Engine, a powerful templating engine and scripting language for cool developers!

## Building the documentation
The documentation is built using [Booyaka](https://github.com/openpeeps/booyaka), a documentation site generator built with Nim.

```
git clone https://github.com/openpeeps/timdocs
booyaka start timdocs
```

Done! The documentation should be live at `http://localhost:3000`

## Contributing
If you want to contribute to the documentation, feel free to fork the repo and submit a pull request. You can also report any issues or suggest improvements by creating a new issue on the [GitHub Issues page](https://github.com/openpeeps/tim/issues).
